建立本地Podspec

這裡記錄如何在本地建立 Podspec 模組

  • 建立方式

    1. 終端機執行 pod lib create TestPodFramework
    2. 依照需求回答一系列問題，最後建立專案
    3. 打開專案會看到 Demo 和 Framework 本身



    1. 替換 Development Pods 內的檔案或程式碼，新增自己要的類別或功能在 Class 資料夾(注意一定要在這)







    1. 重新 pod install，即可直接編譯 Demo 測試功能
      • (記得使用 Git 進行版控，以確保每個成員使用的版本)



  • 使用方式:

    1. 當其他專案要使用時，將 TestProject 和 TestPodFramework 放在同一層



    1. 新增 TestProject 的 podFile，並加上 pod 'TestPodFramework', :path => '../TestPodFramework' 意思就是讀取本地模組



    1. 重新 pod install，最後 import Framekwork 就可以使用
