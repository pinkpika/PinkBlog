Hexo-文章功能

發表於1月 12 2015|更新於2月 6 2022|Web DevelopmentHexo
|閱讀量:

跟文章有關的功能，目前會記錄在此

發表帶有標籤、分類的文章

  1. 新增文章
  1. 編輯 md 檔案，加上 tags 或是 category

新增文章圖片

官方文件 Include-Assets

  1. 預設方式-統一管理
    • 圖片放進 source/images
    • 編輯 md 使用 ![](/images/image.jpg)
  2. 進階方式-個別管理
    • 編輯 _config.yml 開啟 post_asset_folder: true
    • 圖片放進同名文章的資料夾內 _posts/testpost/image.jpg
    • 編輯 md 使用 ![](image.jpg)

語法優化(使用以下語法才可以在首頁顯示)

{% asset_path filename %}
{% asset_img [class names] slug [width] [height] [title text [alt text]] %}
{% asset_link filename [title] [escape] %}

{% asset_img result.jpg 400 "lorem ipsum"%}

例如：

{% asset_img hexo.jpg 400 %}
