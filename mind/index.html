mind

這裡記錄一些成長方法、心智或價值觀的一些概念。

學習成長

精準學習

  • 不要再陷入低效的學習，應學會該領域的臨界知識
  • 臨界知識指的是說可以通用的法則，不只是單單解決某件事
    • ex. 複利效應-任何事都存在複利效應
    • ex. 黃金思維圈-使用why>how>what的思維順序
    • ex. 機會成本-被捨棄的選項中價值最高的那一個

原子學習

  • 兩分鐘法則
  • 四大原則:
    1. 暗示越明顯越好
    2. 動機越誘人越好
    3. 行動越簡單越好
    4. 獎賞越有感越好

高效學習

  1. 遷移法(小事開始學習)
  2. 組塊法(組織各項知識)
  3. 回想法(比二次學習好)
  4. 發散法(輕鬆的狀態更好)
  5. 類比法(可用已知的知識類推)

高效習慣

  1. 清晰目標
  2. 維持活力
  3. 提升重要性
  4. 擴充生產力
  5. 累積影響力
  6. 展現勇氣

改變方法的EAST框架

  1. 簡化事情
    • 看到較為簡單的事，比較有慾望執行
    • ex.事情簡單化、細分化
  2. 增加吸引力
    • 看到厲害的人，自己會想跟他一樣厲害
    • ex.追蹤twitter
  3. 增加社會連結度
    • 看到周遭的人有做，自己也會想做
    • ex.朋友一起做
  4. 增加時效性
    • 看到目標清單或心靈雞湯，當下會想做
    • ex.訂定每周日為重新調整日

深度工作力

  1. 不要等時間找你，要自己找時間(time block)
  2. 下班就認真下班
  3. 追蹤成長的進度(Chain Method)

能力維度

  • 到下一個維度要花80%的資源(時間或錢)

三種職能的人

  • 工匠、領導、行腳商人
  • 要知道自己的產品、技術的優勢、為什麼要這樣做

睡眠相關

  1. R90睡眠法
    • 90分為一個週期
    • 一週有30~35個週期即可
    • 例如4444455
  2. 固定時間起床入眠
  3. 晚上不做激烈運動
  4. 光線暗、溫度低、心平靜

理財相關

  1. 存多少比賺多少重要
  2. 看書能累積財富
  3. 不買車?(消耗財)
  4. 零利率不代表沒利息
  5. 做預算
  6. 提早做計劃可省錢
  7. 越早開始越好(複利)
  8. 窮感富感是相對感

自我介紹

  1. 要有記憶點
  2. 能帶出畫面
  3. 精簡短
  4. 聲調(開心or平穩)
  5. 擅長的事
  6. 用標語的方式

生時間(Google創投的時間管理術)

  • 設計衝刺(Design Sprint)

    1. 星期一：理解目標
    2. 星期二：分開發想
    3. 星期三：選出最佳
    4. 星期四：製作原形
    5. 星期五：用戶測試

  • 四個技巧

    1. 決定生時間要做甚麼
      • 不僅是重要的事，而且是對自己是重要的事
      • 每一天都要有個焦點
    2. 集中能量專注力=>避開分心
    3. 維持活力=>吃食物非食品、重睡眠、重運動
    4. 反省回顧=>做紀錄、追蹤成果

成功人士的七個跡象

  1. 有很多好習慣、少壞習慣
  2. 強大的自我驅動力
  3. 不可動搖的自信
  4. 溝通能力
  5. 具有成長思維
  6. 擅長接受失敗，強大的心智
  7. 超高的情商
